## Welcome to SMP Simulator

The SMP Simulator is a browser-based educational tool that visualizes symmetric multiprocessing systems with 2-4 in-order cores, private L1 instruction and data caches, and MSI cache coherence protocol implementation.

## What You'll Learn

This simulator helps you understand:

- **Parallel Execution**: How multiple cores execute workloads simultaneously and achieve speedup
- **Cache Coherence**: MSI protocol states (Modified, Shared, Invalid) and bus snooping behavior
- **Pipeline Hazards**: Instruction latency, memory access delays, and branch misprediction penalties
- **Branch Prediction**: gshare predictor with global history register and 2-bit saturating counters
- **Performance Metrics**: IPC (instructions per cycle), speedup, MPKI (misses per thousand instructions), and coherence overhead
